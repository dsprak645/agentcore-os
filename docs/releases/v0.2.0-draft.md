# v0.2.0 (Draft)

This is a planning document for the next minor release after `v0.1.0`.

## Themes

- Desktop UX upgrades (resize, tiling, keyboard flow)
- “Workflow Playbooks” you can actually run daily
- Publishing hub reliability (queue, retries, receipts) with BYO connectors
- Project maturity (docs, CI, dependency hygiene)

## Planned changes

### Desktop UX

- Window resize (edges/corners) and size persistence
- Better snap/tiling presets and shortcuts
- Spotlight: recent history + improved command UX

### Playbooks & templates

- Save/share playbooks (steps + app links + prompts)
- Draft templates/snippets and repurposing flows

### Publishing hub

- Queue + retries + backoff
- Richer receipts and connector debugging
- Reference connector recipes for approved services (Make/Zapier/Buffer/Metricool)

## Compatibility

No breaking changes planned (subject to change during implementation).

## Implementation status (in `main`, unreleased)

- Desktop UX: resizable windows + keyboard tiling/restore shortcuts
- Spotlight: local recent apps/commands + `?` help actions
- Playbooks: save/export/import MVP (local-first)
- Publisher: queued dispatch with basic retry/backoff (while Publisher is open)
- Solutions Hub: curated workflow packs + search

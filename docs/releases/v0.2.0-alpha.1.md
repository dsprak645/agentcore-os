# v0.2.0-alpha.1

Release date: **2026-03-11**

This is an **alpha / pre-release** focused on v0.2 UX and workflow completeness. Expect some rough edges.

## Highlights

- Desktop UX: **resizable windows** + keyboard **tiling / maximize / restore**
- Spotlight: local **recent apps/commands** + `?` help actions
- Playbooks: save/export/import runnable SOPs (local-first)
- Solutions Hub: curated workflow packs you can install as Playbooks (with search)
- Publisher: queued dispatch with basic retry/backoff (while Publisher is open)

## Keyboard shortcuts

- `⌘K` / `Ctrl+K`: Spotlight
- `⌘⌥←/→/↑/↓` (or `Ctrl+Alt+←/→/↑/↓`): tile left/right, maximize, restore

## Notes

- Secrets are not stored securely in this demo (browser `localStorage`). Do not use as-is for multi-user production.
- “Auto publish” is intentionally a **connector problem** (official APIs / approved tools via webhooks).
